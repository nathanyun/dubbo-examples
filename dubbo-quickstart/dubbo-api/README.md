## 项目介绍
本模块是通过Dubbo官方提供的DubboBootstrap API来构建应用程序
> 注意: 项目中包含两个main方法, 不支持jar方式运行.

## 项目测试
1. 导入IDEA运行 `DubboBootstrapProvider`的 `main`方法即可启动Dubbo服务提供者.
2. 运行 `DubboBootstrapConsumer`的 `main`方法即可启动Dubbo服务消费者观察控制台日志即可.
